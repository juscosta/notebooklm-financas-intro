# Finanças Introdutórias: Juros Compostos, Inflação (IPCA) e Renda Fixa (Tesouro Direto)

Projeto do desafio: criar um caderno temático no **NotebookLM** com **3 a 5 fontes abertas**, definir objetivos de estudo, elaborar perguntas estratégicas e testar variações de prompts, registrando respostas e referências.

- **Tema escolhido:** Finanças Introdutórias: Juros Compostos, Inflação (IPCA) e Renda Fixa (Tesouro Direto)
- **Data de elaboração:** 2025-12-18
- **Como usar:** siga o passo a passo em [`docs/01_setup_notebooklm.md`](docs/01_setup_notebooklm.md) e replique o caderno no NotebookLM.

## O que tem aqui

- `docs/01_setup_notebooklm.md` — como montar o caderno no NotebookLM (com a curadoria de fontes).
- `docs/02_objetivos_e_plano.md` — objetivos de estudo e plano sugerido (1–2h).
- `docs/03_perguntas_estrategicas.md` — perguntas para guiar a aprendizagem ativa.
- `docs/04_prompts_reutilizaveis.md` — biblioteca de prompts (reutilizáveis).
- `docs/05_resultados_evidencias.md` — registro de testes de prompts + respostas + referências.
- `docs/06_glossario.md` — glossário dos conceitos essenciais.
- `docs/07_resumos_fontes.md` — resumos estruturados por fonte.
- `sources/README.md` — lista de fontes e links (não inclui os PDFs).
