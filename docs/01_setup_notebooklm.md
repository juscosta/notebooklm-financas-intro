# 01 — Montagem do caderno no NotebookLM

> Objetivo: montar um caderno temático com as **5 fontes abaixo**, criando uma base confiável para estudo e revisão.

## 1) Crie o caderno
1. Abra o NotebookLM (Google).
2. Clique em **New notebook** (ou “Novo caderno”).
3. Nomeie como: **Finanças Introdutórias: Juros Compostos, Inflação (IPCA) e Renda Fixa (Tesouro Direto)**.

## 2) Adicione as fontes (3 a 5)
Adicione as fontes **S1–S5** listadas em `sources/README.md`.

### Dica prática
- Para PDFs: use **Upload file** (baixando o PDF do link) ou cole o link se o NotebookLM aceitar URL direta.
- Para páginas HTML: cole o link direto (S2).

## 3) Padronize nomes e tags (recomendado)
No caderno, renomeie as fontes assim:
- `S1_BCB_Gestao_Financas_Pessoais.pdf`
- `S2_IBGE_Inflacao.html`
- `S3_eduCAPES_Matematica_Financeira.pdf`
- `S4_B3_Guia_Renda_Fixa.pdf`
- `S5_Santander_Tesouro_Direto.pdf`

Crie tags (se usar) como:
- `juros`, `juros_compostos`, `inflacao`, `ipca`, `renda_fixa`, `tesouro_direto`, `riscos`, `impostos`

## 4) Crie a “página índice” do seu caderno (no próprio NotebookLM)
Cole este bloco como a primeira nota:

**Índice**
1. Objetivos do estudo
2. Resumos das fontes (S1–S5)
3. Glossário
4. Perguntas estratégicas
5. Prompts reutilizáveis
6. Registro de testes (respostas + referências)

## 5) Regras de estudo (para evitar “resposta bonita, mas errada”)
No NotebookLM, fixe (pin) um prompt de regras:

**Prompt fixo (Regras)**
- Responda **somente** com base nas fontes S1–S5.
- Se algo não estiver nas fontes, diga “não consta nas fontes” e sugira que fonte adicionar.
- Sempre cite **qual fonte** e, quando possível, **trecho/página**.
- Seja didático, com exemplos simples e números redondos.

Marque quando concluir:

- [ ] Caderno criado no NotebookLM

- [ ] Objetivos de estudo definidos (docs/02)

- [ ] Miniguia final com resumos + glossário + prompts (docs/06 + docs/07 + docs/04)
